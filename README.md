 Project Overview

This project presents an end-to-end Chocolate Shipment Analytics Dashboard designed to analyze sales performance, shipment trends, profitability, and regional demand across multiple products and geographies.

Business Objective

-The objective of this project is to help stakeholders:

-Monitor current year vs previous year performance

-Identify top-performing products and salespersons

-Analyze shipment volume distribution

-Track profitability across products

-Understand regional contribution to revenue
